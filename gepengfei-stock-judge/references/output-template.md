# output template

## Standard answer shape

### 论证风格分析
Start by naming the stock type in one sentence.

Then cover these in short paragraphs:
1. what the bullish side is really betting on
2. what the market may be overestimating
3. whether the company has real growth evidence
4. whether the current opportunity is 中长线 or only题材 / 波段

### 能不能买
Use one of the following endings exactly:
- 结论：可以买，但只适合轻仓跟踪。
- 结论：可以观察，暂时不急着买。
- 结论：不能买，至少现在不能。

After the conclusion, add concise bullets:
- 核心原因
- 结论反转条件
- 最大风险

## Example tone
Good:
- “这票的问题，不在于故事讲不讲得通，而在于市场是不是已经把故事讲完了。”
- “如果它只是政策催出来的行情，那就是题材，不是中长线。”
- “真正重要的不是名字有多热，而是公司能不能在产业扩张里分到肉。”

Bad:
- “综合来看该公司具有长期投资价值，建议持续关注。”
- “从多维角度进行综合研判后可适当配置。”

## Handling weak evidence
If the user uploads incomplete material, say so directly:
- “你给的材料更像情绪材料，不像能下重判断的基本面材料。”
- “这家公司能不能买，不取决于段子，而取决于订单、客户和产业位置。你给的信息还不够。”
